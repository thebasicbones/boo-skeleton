# Changelog

All notable changes to boo-package-manager will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-05

### Added
- Initial release of boo-package-manager
- RESTful API with FastAPI framework
- CRUD operations for resources
- Topological sorting for dependency management
- Mongodb database backend
- Resource dependency tracking and validation
- Circular dependency detection
- Cascade delete functionality
- Comprehensive test suite with pytest
- Property-based testing with Hypothesis
- Code quality tools (Black, Ruff, MyPy)
